# MaintPro Landing Page

Landing page institucional simples do **MaintPro**, criada para publicar informações básicas do produto em domínio próprio e apoiar o processo de aprovação de domínio pela **Apple** e pelo **Google**.

## Objetivo

Este repositório existe para hospedar uma página pública com:

- Nome e descrição do MaintPro;
- Identidade da marca;
- Informações institucionais básicas;
- Links úteis (como política de privacidade e contato), quando aplicável.

## Escopo

Este projeto **não** é o app principal do MaintPro.

O foco aqui é apenas uma landing page leve, estática e de fácil manutenção, suficiente para validação de domínio e presença oficial do produto na web.

## Aprovação de domínio (Apple/Google)

Para suporte à aprovação, a página deve permanecer:

- Pública e acessível sem autenticação;
- Coerente com a marca e nome do aplicativo;
- Com conteúdo institucional claro e não enganoso.

## Estrutura atual

- Landing page estática do MaintPro
- Repositório dedicado para publicação simples
